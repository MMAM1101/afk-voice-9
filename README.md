# AFK Voice Bot #9

بوت Discord يجلس في قناة الصوت باستمرار ولا يطلع إلا لما توقفه Railway.

## الأوامر

| الأمر | الوصف |
|-------|-------|
| `!join` | ينضم لفويسك |
| `!play [رابط يوتيوب]` | يشغل صوت من يوتيوب |
| `!stop` | يوقف التشغيل |
| `!leave` | يطلع من الفويس |
| `!ping` | يتحقق من الاتصال |

## متغيرات البيئة على Railway

| المتغير | الوصف | مطلوب |
|---------|-------|-------|
| `DISCORD_TOKEN` | توكن البوت من Discord Developer Portal | ✅ |
| `GUILD_ID` | ID السيرفر | اختياري |
| `VOICE_CHANNEL_ID` | ID قناة الصوت للانضمام تلقائياً عند التشغيل | اختياري |
| `PREFIX` | بادئة الأوامر (افتراضي: `!`) | اختياري |
| `BOT_NUMBER` | رقم البوت للتمييز في اللوق | اختياري |

## الإعداد على Railway

1. **افتح** [railway.app](https://railway.app) وسجل دخول
2. **New Project** → **Deploy from GitHub repo** → اختر هذا المستودع
3. في **Variables** أضف:
   - `DISCORD_TOKEN` = توكن بوتك
   - `GUILD_ID` = ID سيرفرك
   - `VOICE_CHANNEL_ID` = ID قناة الصوت
4. **Deploy** — البوت يشتغل ويجلس في الفويس تلقائياً

## ملاحظات

- البوت يستخدم **yt-dlp** لتشغيل يوتيوب (أكثر مكتبة موثوقة ومُحدَّثة باستمرار)
- يتصل تلقائياً بالفويس عند بدء التشغيل إذا ضبطت `GUILD_ID` و `VOICE_CHANNEL_ID`
- يعيد الاتصال تلقائياً لو انقطع
- لا يطلع من الفويس إلا لما تسويله `SIGTERM` عبر Railway أو تستخدم `!leave`
